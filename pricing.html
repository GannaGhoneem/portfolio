Frequently Asked

Quick answers to common questions

How long does a project take?
Timelines vary by complexity. A Basic web project takes 1–3 days, Balance 3–7 days, PRO 1–2 weeks, and Custom projects are scoped individually. I'll give you a clear timeline upfront.
What payment methods do you accept?
I accept bank transfers, PayPal, Vodafone Cash, and InstaPay. Typically 50% upfront and 50% on delivery for larger projects.
Can I request changes after delivery?
Each plan includes a set number of revision rounds. Additional changes after that are billed at an hourly rate. Custom plans include unlimited revisions during the project phase.
Do you provide source code?
Yes, full source code ownership is transferred to you upon final payment. All code is clean, well-commented, and version-controlled with Git.
Can you work with existing projects?
Absolutely! I can integrate into existing codebases, add features, fix bugs, or refactor legacy code. This typically falls under the Custom plan.
